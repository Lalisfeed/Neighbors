# check-fn

# What is dis?

> A bool function that checks if a variable is a function or not.

# Installation

```
npm i check-fn
```

# Usage

Add this to your `Javascript` file

```javascript 
import { checkFn } from 'check-fn';

checkFn(inputVar); // returns true or false

```
